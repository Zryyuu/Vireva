menurutmu website pemesanan booking hotel itu kalau user harus login itu termasuk mempersulit kah?

Tidak selalu. Login di website booking hotel bisa membantu atau justru mempersulit, tergantung kapan login diminta dan seberapa mudah prosesnya. 🏨

1️⃣ Kalau login diminta di awal

Contoh alur:

Buka website

Harus login dulu

Baru bisa cari hotel

Ini biasanya dianggap mempersulit user ❌ karena:

user belum tahu apakah hotelnya cocok

user belum yakin mau booking

banyak orang langsung keluar dari website

Banyak website besar seperti Booking.com atau Airbnb membiarkan user:

cari hotel

lihat harga

lihat foto

tanpa login dulu.

2️⃣ Kalau login diminta saat akan booking

Contoh alur:

Cari hotel

Lihat detail

Klik Book Now

Baru diminta login

Ini lebih ramah pengguna ✅ karena:

user sudah yakin mau pesan

login hanya untuk menyimpan data pemesanan

3️⃣ Cara yang paling user-friendly

Website modern biasanya pakai:

Guest checkout (booking tanpa akun)

Login dengan Google

Login dengan email OTP

Ini mempercepat proses dan mengurangi hambatan.

✅ Kesimpulan:
Login tidak mempersulit jika diminta di tahap akhir saat booking.
Tetapi jika dipaksa login sebelum melihat hotel, itu bisa mengurangi kenyamanan user dan membuat orang keluar dari website.
